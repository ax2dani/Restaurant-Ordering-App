# 🍔 The K-Town Diner - Restaurant Ordering App

A responsive, interactive food ordering application built as part of the **Scrimba Essential JavaScript Course**. This project demonstrates the ability to transform a Figma design into a functional web application using vanilla JavaScript, focusing on data-driven UI and clean user experiences.

## 🛠️ Features

* **Dynamic Menu Rendering:** Menu items are generated dynamically from a data module.
* **Interactive Shopping Cart:** Add or remove multiple items with real-time total price calculation.
* **Unique Item Tracking:** Uses UUIDs to allow users to remove specific instances of the same item (e.g., removing one of two pizzas).
* **Payment Modal:** A sleek, validated checkout form for processing "payments."
* **Star Rating System:** A post-checkout experience allowing users to rate their meal.
* **Responsive Desktop View:** Optimized for both mobile and desktop browser formats.

## 🌟 Stretch Goals Implemented

* **Dark Mode Toggle:** A fully functional theme switcher using CSS Variables and LocalState logic.
* **Meal Deal Logic:** Automatically applies a **20% discount** when a Hamburger and Beer are added to the same order.
* **Order Completion Feedback:** Personalized "Thank You" message that replaces the cart after a successful checkout.

## 💻 Tech Stack

* **HTML5:** Semantic structure.
* **CSS3:** Flexbox, CSS Variables, and responsive media queries.
* **JavaScript (ES6+):** * ES Modules (`import`/`export`)
* Event Delegation
* `.map()`, `.filter()`, and `.some()` array methods
* Template Literals
* UUID generation
